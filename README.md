# 🎓 CreditCalc – Student Credit Calculator

CreditCalc is a modern and responsive academic credit calculator designed for university and college students. The application allows students to calculate academic credits, monitor module performance, track graduation progress, and manage semester results in one simple dashboard.

The project was developed using HTML, CSS, and JavaScript and stores data locally using the browser’s LocalStorage API.

---

# 📖 Project Overview

Managing academic credits manually can be difficult and time-consuming for students. CreditCalc was created to simplify the process by providing an easy-to-use web application where students can:

- Add academic modules
- Enter marks and credit values
- Calculate passed and failed credits
- Monitor academic performance
- View graduation progress
- Save module information automatically

The system is designed with a clean and professional interface that works across desktop and mobile devices.

# ⚙️ System Functionality

## 1️⃣ Adding Modules
Students can enter:
- Module Name
- Semester
- Credit Value
- Percentage/Mark

The information is stored in an array and displayed dynamically on the page.

---

## 2️⃣ Calculating Credits
JavaScript functions calculate:
- Total registered credits
- Passed module credits
- Failed module credits
- Average percentage

Example:
- Modules with marks above 50% are counted as passed
- Modules below 50% are counted as failed

---

## 3️⃣ Academic Progress
The app calculates progress toward graduation requirements.

Example:
- Required graduation credits: 360
- Current credits earned: 210
- Progress displayed as a percentage

---

## 4️⃣ LocalStorage Integration
All module data is saved automatically using LocalStorage so students do not lose information after refreshing the browser.
